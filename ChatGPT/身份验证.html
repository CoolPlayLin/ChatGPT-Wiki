# 验证方法

您不能同时将多种形式的身份验证放入一个配置中

# 邮箱/密码

  • 不支持使用 Google 和 Microsoft 授权的注册的账户
  • 使用 https://github.com/acheong08/OpenAIAuth 来进行本地验证
  • 允许你获取 session_token

# 会话令牌

  • 在chat.openai.com的 cookie 中找到__Secure-next-auth.session-token
  • 允许你从https://chat.openai.com/api/auth/session获取access_token
  • 支持所有的账户类型
  • 不知道什么时候会失效

# 访问令牌

  • 这是实际用于身份验证的内容，可以在https://chat.openai.com/api/auth/session找到
  • 2周后失效
  • 推荐的身份验证方法
  • 如果您登录到https://chat.openai.com/，然后转到https://chat.openai.com/api/auth/session，就可以找到
